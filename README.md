# Oilspill_Dashboard
Building a dashboard to monitor oil spills across the US

Data:
-  US_oilspills.csv: csv with information about US oilspills with columns:
    -  id 
    -  open_date 
    -  name 
    -  location 
    -  lat 
    -  lon 
    -  threat
    -  commodity 
    -  max_ptl_release_gallons 
    -  field_10 to field_25

-  US Map
